# MOTOF

**Multi-Objective Transportation Optimization Framework**

---

## Vision

MOTOF is an extensible Python framework dedicated to research in
multi-objective transportation optimization.

The framework supports:

- Multi-objective transportation problems
- Evolutionary algorithms
- Exact optimization
- Transportation encodings
- Locality analysis
- Statistical analysis
- Experiment automation
- Reproducible scientific research
